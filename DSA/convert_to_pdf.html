Core Patterns Overview

  1. Two Pointers - Array/string problems with sorted data
  2. Sliding Window - Contiguous subarray/substring problems
  3. Fast & Slow Pointers - Cycle detection in linked lists
  4. Merge Intervals - Overlapping interval problems
  5. Cyclic Sort - Arrays with numbers in given range
  6. In-place LinkedList Reversal - Reversing lists without extra space
  7. Tree BFS - Level-order traversal problems
  8. Tree DFS - Path sum and tree traversal
  9. Two Heaps - Finding median in stream
  10. Subsets - Generating combinations/permutations
  11. Modified Binary Search - Search in sorted/rotated arrays
  12. Top K Elements - Heap-based K element problems
  13. K-way Merge - Merging multiple sorted lists
  14. Dynamic Programming - Optimization and counting problems
  15. Topological Sort - Dependency and scheduling problems
  16. Graph Algorithms - BFS/DFS and connected components
  17. Backtracking - Constraint satisfaction problems
  18. Monotonic Stack - Next greater/smaller element
  19. Union Find - Connected components and cycle detection
  20. Trie - Prefix matching and word search

Key Interview Strategies

The UMPIRE Method

  • Understand - Clarify the problem
  • Match - Identify the pattern
  • Plan - Design your approach
  • Implement - Write clean code
  • Review - Test with examples
  • Evaluate - Analyze complexity
